Theorem 5.13 — Exact F test

Textbook statement

Partition the regressors as [X_1, X_2] and test the block null H_0:\beta_2=0. Under the null, the response has the form y = X_1\beta_1+\varepsilon,\qquad \varepsilon\sim N(0,\sigma^2 I_n). With the usual full-rank assumptions, \sigma^2>0, a positive number of tested restrictions q=\dim(X_2)>0, and positive residual degrees of freedom, Hansen Theorem 5.13 states that F = \frac{(RSS_R-RSS_U)/q}{RSS_U/(n-k)} \sim F_{q,n-k}. If the critical value c has upper-tail probability \alpha under that distribution, the test that rejects when F>c has exact size \alpha.

Proof sketch

Let D = P_{[X_1,X_2]} - P_{X_1}. This projection increment captures the fitted-value contribution of X_2 after controlling for X_1. Lean proves D is symmetric, idempotent, and has rank q. Under the null, \frac{RSS_R-RSS_U}{\sigma^2} = \frac{\varepsilon^\top D\varepsilon}{\sigma^2} \sim \chi^2_q. The denominator statistic is the full-model residual variance component, already covered by Theorem 5.7: \frac{RSS_U}{\sigma^2}\sim\chi^2_{n-k}. The two statistics are independent because D M_{[X_1,X_2]}=0 and the underlying vector is Gaussian. The ratio of independent chi-square variables, each divided by its degrees of freedom, has the Fisher-Snedecor distribution.

The headline theorem reads:

theorem olsFStatistic_hasLaw_classicalFDist
    {Ω : Type*} [MeasurableSpace Ω] {μ : Measure Ω}
    (X₁ : Matrix n k₁ ℝ) (X₂ : Matrix n k₂ ℝ) (β₁ : k₁ → ℝ) {σ2 : ℝ}
    (hσ2 : 0 < σ2) (hq : 0 < Fintype.card k₂)
    (hdf : Fintype.card (Sum k₁ k₂) < Fintype.card n)
    (ε : Ω → EuclideanSpace ℝ n)
    [Invertible (X₁ᵀ * X₁)]
    [Invertible ((Matrix.fromCols X₁ X₂)ᵀ * Matrix.fromCols X₁ X₂)]
    (hε : HasLaw ε (multivariateGaussian 0 ((σ2 : ℝ) • (1 : Matrix n n ℝ))) μ) :
    HasLaw (fun ω => olsFStatistic X₁ X₂ (X₁ *ᵥ β₁ + WithLp.ofLp (ε ω)))
      (classicalFDist (Fintype.card k₂) (Fintype.card n - Fintype.card (Sum k₁ k₂))) μ

Translation notes

  • The formal null is encoded by building the response from X₁ *ᵥ β₁; the right block coefficient is therefore zero by construction.
  • The proof uses Sum k₁ k₂ as the column index type of the unrestricted design. Its cardinality is Fintype.card k₁ + Fintype.card k₂.
  • olsFStatistic_rejection_probability_eq_alpha_classical is the final testing statement: it transports the F law through the upper-tail event {ω | c < F(ω)}.
