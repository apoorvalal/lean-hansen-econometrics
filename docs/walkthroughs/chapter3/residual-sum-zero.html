Equation 3.17 — Residuals sum to zero with an intercept

Textbook statement

For a design matrix X \in \mathbb{R}^{n\times k} and response vector y \in \mathbb{R}^n, when X^\top X is invertible and the constant vector \mathbf{1} lies in the column span of X, OLS residuals sum to zero: \sum_{i=1}^n \hat e_i = 0.

The “intercept” condition is formalized as a vector of coefficients c with Xc=\mathbf{1}.

Proof sketch

The sum of residuals is the residual vector dotted with the all-ones vector: \sum_i \hat e_i = \hat e^\top \mathbf{1}. If \mathbf{1}=Xc, this becomes \hat e^\top Xc = (X^\top\hat e)^\top c. The normal equations say X^\top\hat e=0, so the expression is zero.

Lean statement

The theorem lives in HansenEconometrics/Chapter3LeastSquaresAlgebra.lean:

theorem residual_sum_zero_of_one_mem_colspan
    (X : Matrix n k ℝ) (y : n → ℝ) [Invertible (Xᵀ * X)]
    {c : k → ℝ} (hc : X *ᵥ c = 1) :
    ∑ i, residual X y i = 0

Reading the statement

Lean Math What it gives us
{c : k → ℝ} coefficient vector c A witness that the constant vector is a linear combination of the columns of X.
hc : X *ᵥ c = 1 Xc=\mathbf{1} The intercept / constant-in-column-space assumption. See the Chapter 3 assumption map.
∑ i, residual X y i = 0 \sum_i \hat e_i=0 Residuals have mean zero up to the conventional factor 1/n.

Translation notes

  • Lean writes the all-ones vector simply as 1, because vectors n → ℝ have a pointwise one.
  • The theorem uses a concrete witness c rather than an existential hypothesis. That makes later proofs directly rewrite 1 as X *ᵥ c.
  • This is a consequence of the normal equations, not an extra property of the objective function.

Downstream uses

This is the theorem behind the familiar statement that residuals average to zero in a regression with an intercept. It also records the precise condition needed by later finite-sample decomposition arguments: the all-ones vector must be in the column space of the design.
